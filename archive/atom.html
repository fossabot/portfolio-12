--- title: Atom layout: contribution ---

Atom being one of my favorite text editor, I wanted to contribute to this open source project. I started with a small bugfix for the wrap-guide package (#72). Then a small improvement to the Atom.io website, changing the documentation in the Flight manual (#420). And there is also a Pull Request open on the main editor's repository to add an option for community packages to prevent a tab from being closed (#13812).

Besides contributing to main project I also contributed to the Atom Linter project by adding an intelligent disable function to linter package for HTMLHint (#172) and creating, based on a fork linter-htmlhint, a new linter plugin for htmllint.

On top of that I also contributed to community packages. For example the language-latex package, changing the snippets such that both subpar and \subpar are autocompleted to \subparagraph (#181)
