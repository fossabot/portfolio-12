NervousFish

2017

App for your 📱 to exchange public-keys in a secure manner.

  • Android
  • Bluetooth
  • Gradle
  • Java

Description

I did this project at TU Delft. We created an Android application which can be used to exchange public keys over Bluetooth, NFC or via QR codes to replace or ease Key Signing Parties. We focused on preventing Man in the Middle attacks when Bluetooth is used so users are sure they received the correct key.
