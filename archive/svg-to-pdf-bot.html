svg-to-pdf-bot

2018

A GitHub App built with Probot that converts SVG files into PDF files for Simple Icons.

  • Jest
  • NodeJS
  • Probot

Description

This GitHub bot was started because of an issue created by a contributor of the Simple Icons project (#743). It is hosted with Now.sh and automatically converts any SVG icon committed to the develop branch of the Simple Icons repository and converts it into a PDF version of the icon. Currently, the bot is being tested and it commits the PDF files to a separate repository (simple-icons-pdf) so the behavior of the bot and the PDF files can be evaluated in isolation.
