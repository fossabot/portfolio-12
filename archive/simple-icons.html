I started contributing to the Simple Icons project by submitting a Pull Request for the CSS3 icon (#373), since then I added multiple different icons. Besides that I implemented a relative luminance filter in Jekyll to darken icons on lighter backgrounds (#488), added fuzzy search (#659) and improved the docs on multiple occasions. Over time I became more invested into the project and became one of the maintainers as well.
