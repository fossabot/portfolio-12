SVGOMG

2018

SVGOMG is SVGO's Missing GUI, aiming to expose the majority, if not all the configuration options of SVGO.

Description

SVGOMG is a tool that I frequently use and got to know through the Simple Icon project. I myself and the Simple Icons community were facing an issue regarding the settings of the tool, and wanted a way to reset them. Hence, I went ahead an proposed a reset button in Pull Request #141, which lead to Pull Request #142 in which I collaborated with Jake Archibald (the maintainer of SVGOMG) to get a working reset button for the settings on the website.

You can now use the button on the site! It is at the bottom of the settings panel.
