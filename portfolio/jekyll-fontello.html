--- title: jekyll-fontello layout: project ---

I started with this plugin for this portfolio (which is build using Jekyll) so that I wouldn't have to manually download my webfont from Fontello each time I needed a new icon. During the development of this site I modified and tweaked the script, and finally I decided to release it as a Gem. I also added it to the list of plugins on the Jekyll website (#6757) and the Fontello README.md (#635).
